# Movie Library (Личная кинотека)

**Автор:** Хуранов Мартин

## Описание

Приложение для хранения информации о фильмах с графическим интерфейсом (tkinter).  
Позволяет добавлять фильмы, фильтровать их по жанру и году, а также сохраняет данные в JSON-файл.

## Функциональность

- Добавление фильма (название, жанр, год, рейтинг)
- Валидация: год — целое число, рейтинг от 0 до 10
- Просмотр всех фильмов в таблице
- Фильтрация по жанру (частичное совпадение)
- Фильтрация по году выпуска (точное совпадение)
- Сохранение и загрузка из `movies.json`
- Сброс фильтров

## Установка и запуск

1. Убедитесь, что установлен Python 3.6+.
2. Скачайте файл `movie_library.py`
3. Запустите в терминале:

```bash
python movie_library.py
